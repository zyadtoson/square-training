🔧 إعداد قاعدة البيانات

هذا التطبيق يحتاج Firebase لحفظ البيانات على السحابة حتى يعمل مع الطلاب من أي جهاز. اتبع الخطوات التالية مرة واحدة فقط:

  1. اذهب إلى console.firebase.google.com وسجل دخول بحساب Google
  2. اضغط Create a project (اختر اسم أي اسم) وانتظر حتى ينتهي الإنشاء
  3. من داخل المشروع اضغط على أيقونة </> (Web app) وسجل التطبيق
  4. انسخ كامل كود firebaseConfig الذي يظهر لك
  5. من القائمة الجانبية: Build → Firestore Database → Create database → Start in test mode
  6. عد هنا، الصق الكود في الحقل أدناه واضغط حفظ

💡 فكرة التطبيق - WeDeliver

WeDeliver بيخلي المستخدم يطلب أكل، دواء، أو مشتريات من أي متجر قريب منه. التطبيق فيه مودين:

  • Customer: يتصفح، يطلب، يدفع، يتابع الأوردر
  • Delivery Agent: يشوف الأوردرات، يقبل، يتوجه، يوصّل، يتابع أرباحه

كل تيم بيصمم الموداتين بنفس الـ Visual Style.

🛠️ مراحل العمل

01 — Structure: اكتب هيكل التطبيق — اسم كل شاشة والـ Flow بينهم. حدد إيه المحتوى في كل شاشة. المخرج: فريم في Figma — مش تصميم، بس هيكل نصي واضح.
02 — Mini Design System: اختار الألوان، الخطوط، والـ Grid. سمّي كل لون واستخدامه. حدد أحجام الخطوط والـ Spacing المتكرر.
03 — Components: ابنِ الـ Components الأساسية بـ Auto Layout — Buttons, Input Fields, Cards, Nav Bar, Icons. كل Component يكون reusable ومبني على الـ Design System.
04 — Screens Design: صمم كل الشاشات بالـ Components اللي بنيتها. ابدأ بـ Customer App ثم Delivery Agent App.
05 — Prototype & Delivery: اربط الشاشات في Figma Prototype (Customer: Home → Cart → Checkout → Tracking | Agent: Dashboard → قبول أوردر → تأكيد التوصيل).

📅 متطلبات التسليم ومعايير التقييم

المرحلة / الأيام المطلوب تسليمه معيار التقييم والوزن
يوم 1-2: Structure هيكل التطبيق + الشاشات والـ Flow تنظيم الملف (5%)
يوم 3: Design System الألوان، الخطوط، الـ Grid (Page 1) Mini Design System (20%)
يوم 4-5: Components بناء الـ Components (Page 2) Components + Auto Layout (20%)
يوم 6-8: Screens Customer App (Page 3) + Agent App (Page 4) Visual Design (25%) + اكتمال الشاشات (20%)
يوم 9: Prototype مراجعة + ربط الشاشات (Page 5) Prototype (10%)
يوم 10: التسليم تسليم ملف Figma + عرض 7 دقايق -
